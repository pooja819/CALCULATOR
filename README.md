# Calculator Project

This is a simple web-based calculator project built using HTML, CSS, and JavaScript. It supports basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- **Basic Arithmetic Operations**: Perform addition, subtraction, multiplication, and division.
- **Clear Function**: Reset the calculator display.
- **Chained Calculations**: Continue calculations with the result of previous operations.
- **Responsive Design**: The calculator adjusts to different screen sizes for better usability.

## How to Use

1. **Clone the Repository**: Clone the repository to your local machine using the following command:
    ```sh
    git clone https://github.com/yourusername/calculator-project.git
    cd calculator-project
    ```

2. **Open the HTML File**: Navigate to the `html` directory and open the `index.html` file in your web browser.
    ```sh
    cd html
    open index.html
    ```

3. **Using the Calculator**:
    - **Enter Numbers and Operations**: Click on the calculator buttons to input numbers and operations. The display will update with your inputs.
    - **Perform Calculations**: Click the `=` button to evaluate the expression and see the result.
    - **Clear the Display**: Click the `C` button to reset the calculator and clear the display.

## Additional Features

- **Double Zero and Hundred Buttons**: Quickly add '00' and '100' to your input for convenience.
- **Percentage Button**: Calculate percentages easily.
- **Responsive Layout**: The calculator interface adjusts for different screen sizes, making it user-friendly on both desktop and mobile devices.

## Files and Structure

- `index.html`: The main HTML file that contains the structure of the calculator.
- `calc.css`: The CSS file that styles the calculator.
- `button.css`: Additional CSS file for button styling.
- `calc.js`: The JavaScript file that handles the calculator's functionality.

## Code Explanation

### HTML (index.html)

- Contains the structure of the calculator, including the display input and buttons.
- Links to the CSS and JavaScript files.

### CSS (calc.css and button.css)

- Styles the calculator layout, buttons, and display.
- Uses flexbox for layout and responsive design.

### JavaScript (script.js)

- Handles button clicks and performs arithmetic operations.
- Uses the `eval` function to evaluate expressions.
- Clears the display and handles chained calculations.

## Testing
